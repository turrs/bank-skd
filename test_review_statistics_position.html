// ✅ Mobile First Design:
• Grid: grid-cols-1 md:grid-cols-2 gap-4

🎯 User Experience Flow

// ✅ Optimal User Journey:
1. User membuka halaman review
2. Melihat ringkasan tryout (Session Summary)
3. Review soal satu per satu (Questions Review)
4. Navigasi antar soal (Question Navigation)
5. Analisis statistik per kategori (Statistics) ← Paling bawah
// 💡 Logic:
• Review dulu, analisis belakangan
• Statistik sebagai kesimpulan
• Tidak mengganggu fokus review
